* Mini Project For Semester 6 (2019)

## Functionalities
* Arrange Meetings, add people and notify them
* Record Meeting
* Get Transcripts of the meeting
* Speaker Diarization
* Meeting Summarization
* Task assigning

## Technology Stack
* Android
* Angular
* Firebase
* HTML, CSS, Bootstrap
* IBM Watson STT and NLU API
